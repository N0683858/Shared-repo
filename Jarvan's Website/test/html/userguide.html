USER GUIDE

Intro

This Mechanics simulator is ideal for A-level students that are studying either Physics or Maths. Within this user manual there will be detailed explanations for each step on how to use the simulator and the features there are.

Getting Started

Click the download button and open the simulator after it hs finished installing.


Creating a simulation

1. On the main menu of the product click the button “Create new simulation”.

2. Input the variables as you desire at the top of the page.

3. Click the “run” button to run the simulation with the variables you entered.


Results

1. To see the results of your simulation, click on the button in the bottom right-hand corner.

2. On the results page you can see the results of your simulation in a table and a graph.

3. Click the save results button to save your results in the database.

4. You can either click “Exit” to close the simulation or “Main menu” to go back to the start page.


Loading a Simulation

1. On the main menu click “Load Simulation.

2. The list of simulations saved include the name and variables from that simulation.

4. The selected simulation can either be removed from the database or be loaded.

5. Once loaded the variables will be entered into the simulation and you can run it to watch the ball go down the slope or see the results of the simulation.
